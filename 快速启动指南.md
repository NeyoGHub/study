### 方式二：完整环境（可选）

如果需要完整功能，可以安装依赖：

#### 安装 AkShare 依赖
```bash
# 创建虚拟环境
python3 -m venv akshare-env
source akshare-env/bin/activate

# 安装依赖
pip install akshare pandas

# 运行完整版脚本
python3 scripts/01_basic_usage_fixed.py
```

#### 安装 Docker（FastAPI）
```bash
# 安装 Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER
newgrp docker

# 重新运行检查脚本
python3 scripts/01_basic_setup_fixed.py
```
