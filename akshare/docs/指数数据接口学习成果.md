## 🔬 技术要点总结

### 1. 数据源特点

| 数据源 | 优势 | 劣势 |
|--------|------|------|
| **新浪财经** | 稳定可靠，覆盖面广 | 指数历史数据需通过东方财富接口 |
| **东方财富** | 指数历史数据完整 | 速度较慢（有进度条） |
| **中证指数** | 权威准确 | 仅支持中证系列指数 |

### 2. 关键代码模式

```python
# 模式1: 获取指数历史数据（推荐）
df = ak.index_zh_a_hist(symbol="000300", period="daily",
                         start_date="20250101", end_date="20260507")

# 模式2: 获取指数实时行情
spot = ak.stock_zh_index_spot_sina()

# 模式3: 手动计算申万行业涨跌幅
realtime_sw = ak.index_realtime_sw()
realtime_sw['涨跌幅'] = (realtime_sw['最新价'] - realtime_sw['昨收盘']) / realtime_sw['昨收盘'] * 100
```

### 3. 技术指标计算要点

```python
# 均线
df['MA5'] = df['收盘'].rolling(window=5).mean()

# MACD
ema_fast = df['收盘'].ewm(span=12).mean()
ema_slow = df['收盘'].ewm(span=26).mean()
df['DIF'] = ema_fast - ema_slow
df['DEA'] = df['DIF'].ewm(span=9).mean()
df['MACD'] = 2 * (df['DIF'] - df['DEA'])

# RSI
delta = df['收盘'].diff()
gain = delta.where(delta > 0, 0)
loss = (-delta).where(delta < 0, 0)
avg_gain = gain.rolling(window=14).mean()
avg_loss = loss.rolling(window=14).mean()
rs = avg_gain / avg_loss
df['RSI'] = 100 - (100 / (1 + rs))

# 布林带
df['BOLL_MID'] = df['收盘'].rolling(window=20).mean()
std = df['收盘'].rolling(window=20).std()
df['BOLL_UP'] = df['BOLL_MID'] + 2 * std
df['BOLL_DN'] = df['BOLL_MID'] - 2 * std
```

## 💡 重要发现

### 发现1: 指数数据不需要复权

与个股不同，指数数据本身就是价格指数，已经考虑了成分股变动、分红等因素，**不需要进行复权处理**。

### 发现2: 申万行业涨跌幅需要手动计算

`index_realtime_sw()` 接口返回的数据中**不含涨跌幅字段**，需要通过最新价和昨收盘手动计算。

### 发现3: 接口命名变化

一些旧版接口在新版AkShare中已改名或移除：
- `index_sw` → 不存在，应使用 `index_hist_sw`
- `index_yh` → 不存在，应使用 `index_yw`
- `index_stock_cons` → 不稳定，推荐使用 `index_stock_cons_sina`

### 发现4: 指数代码格式

- A股指数代码：6位数字，如 `000300`（沪深300）、`399006`（创业板指）
- 港股指数代码：字母代码，如 `HSI`（恒生指数）
- 不需要加市场前缀（如 sh, sz）

### 发现5: 数据获取速度差异

- `index_zh_a_hist`：较慢（约20-30秒），有进度条
- `stock_zh_index_spot_sina`：较快（约2秒）
- `index_realtime_sw`：中等（约18秒）
- `index_stock_cons_sina`：较快（约2秒）

## 📝 学习心得

1. **系统化学习效率高**: 按照5步标准流程（调研→测试→分析→示例→文档），每个阶段产出明确
2. **多数据源容错**: 新浪财经是AkShare的核心稳定数据源，东方财富和腾讯作为备用
3. **标准化模板有用**: 统一的研究模板确保了每个数据类别的研究质量一致

5. **数据可视化重要**: 图表比数据表格更直观，金叉死叉信号图能快速识别交易机会
