## 🎯 核心接口

### 1. 历史K线数据 - stock_zh_a_daily
```python
import akshare as ak

# 获取浦发银行历史数据（前复权）
df = ak.stock_zh_a_daily(symbol="sh600000", adjust="qfq")
```

**评价**: ⭐⭐⭐⭐⭐
- 数据完整详细
- 支持复权
- 适合技术分析

---

### 2. 实时行情数据 - stock_zh_a_spot
```python
import akshare as ak

# 获取全市场实时行情
df = ak.stock_zh_a_spot()

# 涨幅榜TOP10
top_gainers = df.nlargest(10, '涨跌幅')
print(top_gainers)
```

**评价**: ⭐⭐⭐⭐⭐
- 数据全面
- 实时性好
- 支持多维度分析

### 3. 分钟级数据 - stock_zh_a_minute

import akshare as ak

## 🔬 关键技术突破

### 1. 数据源优先级研究
经过深度源码分析和实际测试，发现AkShare的核心数据源优先级：

| 优先级 | 数据源 | 接口 | 状态 |
|-------|--------|------|:----:|
| 1 | 新浪财经 | stock_zh_a_daily | ✅ 稳定 |
| 2 | 新浪财经 | stock_zh_a_spot | ✅ 稳定 |
| 3 | 新浪财经 | stock_zh_a_minute | ✅ 稳定 |
| 4 | 腾讯财经 | stock_zh_a_hist_tx | ✅ 备用 |
| - | 东方财富 | stock_zh_a_hist | ❌ 不可用 |

### 2. 中文字体显示问题解决
**问题**: matplotlib无法正确显示中文字符
**解决方案**:
```python
import matplotlib.font_manager as fm
font_path = '/usr/share/fonts/opentype/noto/NotoSansCJK-Regular.ttc'
chinese_font = fm.FontProperties(fname=font_path, size=12)
# 在所有文本元素中使用
ax.set_title('标题', fontproperties=chinese_font)
ax.legend(prop=chinese_font)
```
**效果**: 完全消除中文字体警告，所有中文正确显示

### 3. 金叉死叉信号识别标准化
```python
# 标准识别方法
golden_cross = (df['MA5'] > df['MA10']) & (df['MA5'].shift(1) < df['MA10'].shift(1))
death_cross = (df['MA5'] < df['MA10']) & (df['MA5'].shift(1) > df['MA10'].shift(1))
```
