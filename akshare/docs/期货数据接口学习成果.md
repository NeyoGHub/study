## 🔬 技术要点

### 主力合约代码规则
- 格式: `品种代码 + 0`（如 RB0, CU0, V0, C0）
- 可查 `futures_display_main_sina()` 获取完整列表

### 双数据源策略
```python
# 新浪（推荐，稳定）
df = ak.futures_main_sina(symbol="RB0")

# 东方财富（当前不可用）
df = ak.futures_hist_em(symbol="螺纹钢主连")  # ❌
```

### 期现价差分析
`futures_spot_price_daily` 返回的 `dom_basis_rate` 是主力合约基差率：

| 字段 | 说明 |
|------|------|
| spot_price | 现货价格 |
| dominant_contract_price | 主力合约价格 |
| dom_basis_rate | 基差率（负值=期货升水） |
