## 概述

AkShare 提供丰富的债券数据接口，涵盖国债收益率、债券指数、可转债、中美利率对比等。数据源主要来自中债登、新浪财经、集思录等。

### 支持的债券数据类型

| 数据类型 | 说明 | 数据量 |
|---------|------|--------|
| 国债收益率曲线 | 多期限收益率（3月-30年） | 738条 |
| 中美收益率 | 中美各期限收益率对比 | 9251条 |
| 债券指数 | 国债/综合/新综合指数 | 4584~6087条 |
| 可转债行情 | 实时行情 + 日线 | 350只 |
| 可转债数据 | 可转债基本信息（申购/转股/评级） | 1012只 |
| 可转债深度数据 | 集思录数据（23维度） | 30只 |
| 国债期货 | 中国/美国国债期货 | 各1000条 |

## 国债收益率曲线

```python
import akshare as ak

# 获取国债收益率曲线数据
df_yield = ak.bond_china_yield()
print(f"共 {len(df_yield)} 条收益率数据")

# 筛选中债国债收益率曲线
treasury_yield = df_yield[df_yield['曲线名称'] == '中债国债收益率曲线']
print(treasury_yield.iloc[-1])
```

**返回字段**: `曲线名称`, `日期`, `3月`, `6月`, `1年`, `3年`, `5年`, `7年`, `10年`, `30年`

包含多种收益率曲线：中债国债收益率曲线、中债商业银行普通债收益率曲线(AAA)、中债中短期票据收益率曲线(AAA)等。

## 中美收益率对比

```python
# 获取中美收益率对比数据
df_rate = ak.bond_zh_us_rate()

# 查看最新中美10年期国债收益率差
latest = df_rate.iloc[-1]
spread = latest['中国国债收益率10年'] - latest['美国国债收益率10年']
print(f"中美10Y利差: {spread:.2f}%")
```

**返回字段**: `日期`, `中国国债收益率2年/5年/10年/30年`, `美国国债收益率2年/5年/10年/30年`, `中国GDP年增率`, `美国GDP年增率`

数据从1990年至今，可用于分析中美利差走势。

## 可转债数据

### 可转债实时行情

```python
# 获取可转债实时行情（350只）
df_cov_spot = ak.bond_zh_hs_cov_spot()

# 按涨跌幅排序
top_gainers = df_cov_spot.nlargest(10, 'changepercent')
top_losers = df_cov_spot.nsmallest(10, 'changepercent')
```

### 集思录可转债数据

```python
# 获取集思录可转债数据（30只，23个维度）
df_cb = ak.bond_cb_jsl()

# 常用指标：现价、涨跌幅、转股溢价率、到期税前收益、双低
print(df_cb[['代码', '转债名称', '现价', '转股溢价率', '到期税前收益', '双低']])

# 双低策略筛选（双低 = 价格 + 溢价率*100，越低越有投资价值）
df_cb_sorted = df_cb.sort_values('双低')
print("双低排名前5:", df_cb_sorted[['转债名称', '双低', '到期税前收益']].head())
```

## 常见问题

### Q1: bond_zh_hs_spot 失败？

A: 该接口偶发 JSONDecodeError，是数据源问题。可转债行情可用 `bond_zh_hs_cov_spot` 替代。

### Q2: 可转债代码格式？

A: 可转债代码为6位数字，沪市以11开头（如113527），深市以12开头（如123456）。使用时可能需要 `sh`/`sz` 前缀。

### Q3: 双低策略是什么？

A: 双低 = 转债价格 + 转股溢价率 × 100。值越低代表转债估值越低，是经典的可转债投资策略。

### Q4: 数据更新频率？

A: 收益率曲线每日更新，可转债行情实时更新，集思录数据每日更新。
