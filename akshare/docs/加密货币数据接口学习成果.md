# 加密货币数据接口学习成果

> 学习日期: 2026-05-17
> 项目: AkShare 数据接口研究（阶段5.2 - 补充）

## 一、学习收获

### 1.1 掌握 CoinGecko API 体系

系统化掌握 CoinGecko Demo API 的全部公开端点，涵盖：

- **简单价格** (simple/price / token_price / supported_vs_currencies)
- **币种数据** (list / markets / id / tickers / history / market_chart / ohlc)
- **交易所数据** (exchanges / tickers / derivatives)
- **全局市场** (global / defi)
- **通用数据** (asset_platforms / exchange_rates / search / trending)
- **NFT 数据** (list / id)
- **公开持仓** (companies / entities)

30+端点全部测试通过，验证了 CoinGecko 作为免费加密货币数据源的可靠性和完整性。

### 1.2 理解 AkShare 体系局限性

AkShare 在 A股/基金/期货等**国内市场**数据上表现出色（1073个API），但在**国际市场**特别是加密货币领域存在明显短板：

- 仅3个加密API，覆盖面极窄
- 无 OHLCV 历史数据
- 无市场排名和全局指标
- 数据新鲜度不稳定

结论：**加密货币数据应使用 CoinGecko API 作为独立数据源**，与 AkShare 互补使用。

### 1.3 完整技术分析能力

利用 CoinGecko 提供的 OHLCV 数据，完成了完整的技术分析链：

```
数据获取 → K线绘制 → 均线系统 → MACD → RSI → 布林带 → 信号识别 → 可视化
```

实现了金叉死叉自动识别、超买超卖判断、多币种对比等量化分析功能。

### 1.4 数据可视化能力

使用 Matplotlib 生成4张专业图表：
1. 价格走势 + 均线 + 金叉死叉
2. MACD 指标
3. RSI + 布林带
4. BTC vs ETH 对比

所有图表支持中文显示（Noto Sans CJK字体）。

## 二、关键技术点

### 2.1 CoinGecko API 特点

```
BASE_URL = https://api.coingecko.com/api/v3
限频: 10-30次/分钟（免费层）
鉴权: 无需API Key
返回: JSON格式
```

**优点**：
- 完全免费，无需注册
- 17,000+ 币种，覆盖最广
- 支持 OHLCV、市值、交易量等核心金融数据
- 国内网络可直接访问

**缺点**：
- 免费层限频严格（10-30次/分）
- 无 WebSocket 实时推送
- 部分高级功能需 Pro API（付费）

### 2.2 OHLCV 数据要点

| 参数 | 值 | 说明 |
|:----:|:--:|------|
| days=1 | 分钟级K线 | 粒度最细 |
| days=7-30 | 4小时K线 | 适合中期分析 |
| days=90+ | 日K线 | 适合长期趋势 |
| vs_currency | usd/cny/eur | 计价货币 |

注意：CoinGecko 的 OHLCV 是**聚合数据**（聚合多家交易所），而非单一交易所数据。这与交易所API（如Binance/OKX）返回的单一市场数据不同。

### 2.3 与 CCXT 的对比

| 对比项 | CoinGecko API | CCXT + OKX |
|:------:|:-------------:|:----------:|
| 币种覆盖 | 17,405（最广） | 交易所上线币种 |
| 数据聚合 | 多交易所聚合 | 单交易所 |
| OHLCV周期 | 4小时/日K | 1m/5m/1h/1d等 |
| 限频 | 10-30次/分 | 无严格限制 |
| 安装 | 无需（HTTP API） | pip install ccxt |

选择建议：
- 需要**全市场数据**（市值排名、全局指标、币种信息）→ CoinGecko
- 需要**高频K线**（1分钟、5分钟）→ CCXT + OKX
- 需要**两者结合** → CoinGecko 做宏观分析，CCXT 做交易数据

## 四、知识扩展

### 4.1 数字货币数据类型体系

```
┌──────────────────────────────────────────────┐
│         数字货币数据生态                       │
├───────────┬──────────┬───────────┬────────────┤
│  市场数据  │ 链上数据 │  交易所   │  衍生品    │
├───────────┼──────────┼───────────┼────────────┤
│ 价格/OHLCV │ 地址/交易 │ 现货交易   │ 期货/永续   │
│ 市值/排名  │ Gas/活跃 │ 流动性    │ 未平仓量   │
│ 交易量     │ DeFi/TVL │ 深度      │ 资金费率   │
│ 趋势/热度  │ NFT      │ 费率      │ 合约类型   │
└───────────┴──────────┴───────────┴────────────┘
```

CoinGecko 主要覆盖市场数据和交易所数据层。

### 4.2 免费数据源对比

| 数据源 | 覆盖 | OHLCV | 无需Key | 国内访问 |
|:------:|:----:|:-----:|:-------:|:--------:|
| CoinGecko | ⭐⭐⭐⭐⭐ | ✅ | ✅ | ✅ |
| CoinPaprika | ⭐⭐⭐⭐ | ✅ | ✅ | ✅ |
| CCXT+OKX | ⭐⭐⭐ | ✅ | ✅ | ⚠ 需VPN |
| Binance API | ⭐⭐⭐⭐ | ✅ | ❌需Key | ❌ 451禁止 |

### 4.3 API设计模式

CoinGecko 的 RESTful API 设计值得学习：
- 资源路径清晰：`/coins/{id}/ohlc`
- 查询参数统一：`vs_currency`, `days`, `per_page`
- 响应格式一致：JSON，错误码明确
- 缓存友好：支持 ETag/Last-Modified
