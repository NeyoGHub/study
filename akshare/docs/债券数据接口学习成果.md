## 🔬 关键技术要点

### 1. 收益率曲线接口

`bond_china_yield()` 无参数，返回多条收益率曲线。筛选中债国债收益率曲线的方法：

```python
df = ak.bond_china_yield()
treasury = df[df['曲线名称'] == '中债国债收益率曲线']
latest = treasury.iloc[-1]
```

### 2. 可转债双低策略

集思录可转债数据包含双低指标，是经典的可转债投资参考：

```python
df = ak.bond_cb_jsl()
# 双低 = 转债价格 + 转股溢价率 × 100
df_sorted = df.sort_values('双低')
```

### 3. 债券指数名称

| 接口 | 指数名称 | 数据源 |
|------|---------|--------|
| `bond_treasury_index_cbond` | 中债-国债总指数 | 中债登 |
| `bond_composite_index_cbond` | 中债-综合指数 | 中债登 |
| `bond_new_composite_index_cbond` | 中债-新综合指数 | 中债登 |

### 4. 中美利差分析

`bond_zh_us_rate` 返回的数据包含中美各期限收益率，可以计算利差：

```python
df = ak.bond_zh_us_rate()
df['利差'] = df['中国国债收益率10年'] - df['美国国债收益率10年']
```

### 5. 可转债代码规则

- 沪市可转债: 11xxxx（如 113527）
- 深市可转债: 12xxxx（如 123456）
- 查询时需要加 `sh`/`sz` 前缀

## 💡 重要发现

### 发现1: 债券数据源多元化

与股票（新浪为核心）和基金（东方财富为核心）不同，债券数据源非常分散：
- 收益率/指数 → 中债登
- 可转债行情 → 新浪财经
- 可转债深度 → 集思录
- 期货 → 新浪
- 中美对比 → 宏观经济数据

### 发现2: 集思录可转债数据质量高

`bond_cb_jsl` 虽然只返回30只，但 **23个数据维度** 非常丰富，包含双低、转股溢价率、到期税前收益等投资指标，可直接用于策略分析。

### 发现3: 债券数据稳定性好

14个接口中13个可用（92.9%成功率），是所有阶段中稳定性最高的。
