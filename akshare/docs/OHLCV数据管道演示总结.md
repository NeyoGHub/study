# OHLCV 数据管道演示总结

> 演示日期: 2026-05-17
> 数据源: OKX (CCXT) + CoinGecko API

---

## 一、背景

AkShare 项目原有阶段5.2仅完成加密货币数据源的调研和测试，但未建立可供量化交易使用的持续数据管道。本阶段(9.1)在此基础上，构建了**双源融合的 OHLCV 数据管道**，为后续策略回测和实盘交易奠定数据基础。

---

## 三、数据管道能力

### 3.1 多币种覆盖

覆盖5个主流币种（可扩展）：
- BTC/USDT、ETH/USDT、SOL/USDT、XRP/USDT、DOGE/USDT

### 3.2 多时间周期

单次运行获取每个币种的：
- 1小时K线 (120条)
- 4小时K线 (120条)  
- 日K线 (90-365条)

### 3.3 技术分析

基于OHLCV数据计算10+技术指标：

| 指标 | 参数 | 用途 |
|:----:|:----:|:----:|
| MA10/MA30/MA60 | 10/30/60日 | 趋势判断 |
| MACD | 12,26,9 | 趋势动量 |
| RSI(14) | 14日 | 超买超卖 |
| 布林带 | 20,2 | 波动率通道 |
| ATR(14) | 14日 | 波动率度量 |
| 金叉/死叉 | MA10×MA30 | 买卖信号 |

| 国内网络 | ✅ | ✅ (OKX, CG) |
