## 关键发现

1. **ETF期权代码格式**: 8位数字（如10011381），通过 `option_sse_codes_sina` 获取
2. **希腊字母**: Delta/Gamma/Theta/Vega + 隐含波动率，单只期权查询
3. **东方财富期权接口**: `option_current_em` 等当前不可用，新浪接口稳定
4. **股指期权**: 中金所三大指数期权均可用，含看涨/看跌双边报价
