## 🎯 研究流程（5步标准流程）

### 第1步：数据源调研
- 搜索该数据类别的AkShare接口
- 确认数据源、接口参数、返回格式
- 评估数据质量和稳定性
- 制定多数据源容错机制

### 第2步：接口测试
- 编写测试脚本验证接口可用性
- 测试不同参数组合
- 处理异常情况和网络问题
- 生成测试数据样本

### 第3步：数据分析
- 分析数据字段和统计特性
- 实现相关技术指标计算
- 进行数据质量评估
- 发现数据特点和规律

### 第4步：示例开发
- 编写完整的演示脚本
- 包含金叉死叉信号识别（适用于价格类数据）
- 实现策略回测框架
- 生成可视化图表

### 第5步：文档整理
- 按标准格式编写3份中文文档
- 记录研究过程和发现
- 整理使用指南和最佳实践

## 💻 脚本要求

### 演示脚本标准要求
1. **功能完整性**: 包含数据获取、分析、可视化、导出
2. **金叉死叉信号**: 适用于价格类数据，必须包含信号识别功能
3. **技术指标计算**: 实现相关技术指标（MA、RSI、MACD等）
4. **策略回测**: 提供简单策略框架和收益计算
5. **可视化输出**: 生成高质量的PNG图表
6. **数据导出**: 生成CSV格式数据文件

### 代码规范
```python
# 必须的导入
import matplotlib.pyplot as plt
import matplotlib.font_manager as fm
import pandas as pd
import numpy as np

# 必须的中文字体设置
def setup_chinese_font():
    font_path = '/usr/share/fonts/opentype/noto/NotoSansCJK-Regular.ttc'
    return fm.FontProperties(fname=font_path, size=12)

chinese_font = setup_chinese_font()

# 在所有文本元素中使用fontproperties参数
ax.set_title('标题', fontproperties=chinese_font)
ax.set_xlabel('X轴标签', fontproperties=chinese_font)
ax.set_ylabel('Y轴标签', fontproperties=chinese_font)
ax.legend(prop=chinese_font)
```
