> 演示日期: 2026-05-17
> 数据源: CoinGecko API (Demo Plan, 免费, 无需Key)

---

## 一、背景与动机

AkShare 自带的加密货币接口数量极少（仅3个），且数据覆盖有限：
- `crypto_js_spot` — 约30个USDT交易对实时行情
- `crypto_bitcoin_hold_report` — 比特币持仓报告
- `crypto_bitcoin_cme` — CME比特币数据

**痛点**: 没有历史OHLCV数据、没有市值排名、没有全局市场数据、没有技术分析能力，无法满足量化交易研究需求。

**解决方案**: 替换为 **CoinGecko API**，覆盖30+公开端点，12000+币种，完全免费。

## 七、使用建议

1. **优先使用 CoinGecko API** 作为主要数据源
2. **AkShare 原生接口**作为备用或补充参考
3. **推荐场景**：
   - 量化交易研究 → CoinGecko OHLCV
   - 市场分析报告 → CoinGecko 全局+分类数据
   - 监控面板 → CoinGecko simple/price
   - NFT研究 → CoinGecko NFT端点
4. **注意限频**: 建议每次请求后 `time.sleep(2)`
5. **国内用户**: 可直接访问，无需VPN
