## 实际应用场景

### 场景1：市场监控

**需求**: 监控加密货币市场实时价格

**实现**:
```python
import akshare as ak
import time

# 定期获取实时行情
while True:
    crypto_spot_df = ak.crypto_js_spot()

    # 获取BTC价格
    btc_data = crypto_spot_df[crypto_spot_df['交易品种'] == 'BTCUSD']
    if len(btc_data) > 0:
        btc_price = btc_data['最近报价'].values[0]
        change = btc_data['涨跌幅'].values[0]
        print(f"BTC价格: ${btc_price:,.2f}, 变化: {change:+.2f}%")

    time.sleep(60)  # 每分钟更新
```

### 场景2：持仓分析

**需求**: 分析比特币持仓者分布和机构资金流向

**实现**:
```python
import akshare as ak

# 获取持仓报告
holdings_df = ak.crypto_bitcoin_hold_report()

# 重点关注大额持仓（>10000 BTC）
large_holders = holdings_df[holdings_df['持仓量'] > 10000]

# 按类型分析
institutional = holdings_df[holdings_df['分类'].isin(['上市公司', 'ETF', '政府机构'])]

print("=== 比特币持仓分析 ===")
print(f"总持仓量: {holdings_df['持仓量'].sum():,.0f} BTC")
print(f"大额持仓者(>10K): {len(large_holders)}个")
print(f"机构持仓占比: {institutional['持仓量'].sum()/holdings_df['持仓量'].sum():.2%}")

# 重点关注MicroStrategy
msft = holdings_df[holdings_df['公司名称-中文'].str.contains('MicroStrategy')]
if len(msft) > 0:
    print(f"\nMicroStrategy持仓: {msft['持仓量'].values[0]:,.0f} BTC")
    print(f"持仓成本: ${msft['持仓成本'].values[0]:,.2f}")
```

### 场景3：跨市场对比

**需求**: 对比不同交易所的价格差异

**实现**:
```python
import akshare as ak
import pandas as pd

# 获取实时行情
crypto_spot_df = ak.crypto_js_spot()

# 提取BTCUSD相关数据
btc_usd = crypto_spot_df[crypto_spot_df['交易品种'] == 'BTCUSD'].copy()

# 计算价格统计
price_stats = btc_usd['最近报价'].describe()

print("=== 跨交易所BTC价格对比 ===")
print(f"平均价格: ${price_stats['mean']:,.2f}")
print(f"最高价格: ${price_stats['max']:,.2f} ({btc_usd.loc[btc_usd['最近报价'].idxmax(), '市场']})")
print(f"最低价格: ${price_stats['min']:,.2f} ({btc_usd.loc[btc_usd['最近报价'].idxmin(), '市场']})")
print(f"价格极差: ${price_stats['max'] - price_stats['min']:,.2f}")
print(f"价格波动: {(price_stats['max'] - price_stats['min']) / price_stats['mean']:.2%}")
```

## 常见问题

### Q1: 为什么加密货币数据量这么少？

**A**: AkShare的加密货币接口主要提供市场概览和持仓分析，不是专业的加密货币API。如果需要更详细的数据，建议使用CCXT等专业API。

### Q2: 持仓报告的准确性如何？

**A**: 持仓报告基于公开数据，包括上市公司财报、ETF持仓报告、政府机构公开信息。数据准确性较高，但可能存在更新延迟。

### Q3: 实时行情的更新频率如何？

**A**: 实时行情的更新频率取决于数据源。根据测试结果，当前时间戳显示为2023年，可能不是真正的实时数据。

### Q4: 如何获取更多币种的数据？

**A**: AkShare主要提供主流币种。如果需要更多币种，建议结合CCXT等专业API使用。
