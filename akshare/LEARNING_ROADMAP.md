# AkShare 数据接口学习路线图

**最后更新**: 2026-05-09
**当前阶段**: 阶段3.1 - 期货数据
**整体进度**: 35%（5/14阶段）

## 学习目标

系统掌握AkShare各类金融数据接口，构建完整的金融数据分析能力。

AkShare 共有 **1073个公开API**，覆盖14大类别。本路线图基于源码分析规划最优学习顺序。

#### → 阶段3.1：期货数据（当前进行中）

- **预计时间**: 2天

- **学习重点**:

- **输出成果**:

## 学习方法论

### 每个阶段的标准流程

1. **数据源调研** — 查阅官方文档，了解数据源和接口
2. **接口测试** — 编写 `sX.Y_test_*.py` 测试脚本验证可用性
3. **数据分析** — 分析数据质量、完整性和特点
4. **示例开发** — 编写 `sX.Y_demo_*.py` 含可视化+CSV输出
5. **文档整理** — 3份标准文档（使用指南、演示总结、学习成果）

### 标准化输出

每个阶段完成后必须产生：
- `scripts/sX.Y_test_*.py` — 接口测试脚本
- `scripts/sX.Y_demo_*.py` — 综合示例脚本
- `docs/AkShare[类别]数据接口使用指南.md`
- `docs/[类别]数据接口演示总结.md`
- `docs/[类别]数据接口学习成果.md`
- `output/data/[类别]/` — CSV数据文件
- `output/plots/[类别]/` — PNG图表文件
