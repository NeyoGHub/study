## 快速开始

### 环境搭建

```bash

pip install -r requirements.txt

# 或使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

```python

python scripts/01_basic_usage.py
```

## 常见问题

### Q1: 如何处理依赖安装问题？

A: 建议使用虚拟环境，并指定国内镜像源：
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或 venv\Scripts\activate  # Windows

pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
