## 技术栈详解

### 后端技术
- **FastAPI**: 现代化的 Python Web 框架
- **SQLModel**: 基于 Pydantic 和 SQLAlchemy 的 ORM
- **PostgreSQL**: 关系型数据库
- **Pydantic**: 数据验证和设置管理
- **Alembic**: 数据库迁移工具
- **Pytest**: 测试框架
- **Playwright**: 端到端测试

### 前端技术
- **React 18**: 用户界面库
- **TypeScript**: JavaScript 的超集
- **Vite**: 前端构建工具
- **Tailwind CSS**: 实用优先的 CSS 框架
- **shadcn/ui**: React 组件库
- **Playwright**: 端到端测试

### 部署技术
- **Docker**: 容器化应用
- **Docker Compose**: 服务编排
- **Traefik**: 反向代理和负载均衡

## 故障排除

### 常见问题

#### 1. Docker 启动失败
**解决方案**: 检查Docker服务状态，确保Docker守护进程运行

#### 2. 端口占用
**解决方案**: 检查端口占用情况，修改 Docker Compose 配置中的端口映射

#### 3. 数据库连接失败
**解决方案**: 检查 PostgreSQL 容器状态，确认数据库已启动

#### 4. 依赖安装失败
**解决方案**: 检查网络连接，尝试使用国内镜像源
